# 今日总览布局优化设计

日期：2026-07-13
状态：已确认

## 目标

在不改变总览数据规则和业务操作的前提下，重新组织“今日总览”的视觉层级，解决当前双列布局拥挤、空状态占位过大和各模块样式不统一的问题。

## 已确认设计

- 页面采用纵向层级：本周进度 → 今日任务 → 逾期任务 → 岗位下一步行动 → LeetCode 复习提醒。
- 今日任务作为主模块占满整行，保留现有快速新建、任务完成和取消完成功能。
- 其余三个辅助模块分别占满整行，空数据时仅显示一行紧凑提示，不再使用大面积空状态。
- 各模块统一使用白色卡片、边框、圆角和内边距；内容区扩大并居中，减少宽屏下左侧拥挤、右侧空旷。
- 今日任务为空时只保留一个“新建任务”入口，去掉重复按钮。
- 不修改任务、岗位、LeetCode 的数据筛选规则、Context、DAL 或持久化结构。

## 验收标准

1. 今日任务在本周进度之后占满独立整行。
2. 逾期、岗位行动和复习提醒按顺序分别显示在今日任务下方。
3. 今日任务空状态只出现一个“新建任务”入口。
4. 三个辅助模块空状态为紧凑单行提示。
5. 有数据时的任务勾选、生成岗位任务和完成复习行为保持不变。
6. 客户端测试、类型检查和生产构建通过。
